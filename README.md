# ACME Azure to CyberArk ISP – Connecting Guide

This repository contains a step-by-step guide for connecting the ACME Microsoft Entra tenant and its Azure resources to the CyberArk Identity Security Platform (ISP).

The guide is intended for **Admin B** (Azure/Entra Global Administrator) and covers the manual Azure Portal actions that are equivalent to the CyberArk automation scripts. It sets up the app registrations, federated credentials, custom roles, and role assignments required by two CyberArk services: **Connect Cloud Environments (CCE)** and **Secure Cloud Access (SCA)**.

See [ACME-Azure-Connecting-AdminB-Guide.md](./ACME-Azure-Connecting-AdminB-Guide.md) for the full instructions.
